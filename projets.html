Projets

Rendu de Monnaie

Programme Python calculant automatiquement la monnaie à rendre selon le montant à payer, la somme donnée par le client et les billets ou pièces disponibles dans la caisse.

Ce projet m'a permis d'apprendre à :

  • manipuler des entrées utilisateur avec input ;
  • contrôler les erreurs de saisie et les montants impossibles ;
  • utiliser des listes et des dictionnaires pour représenter une caisse ;
  • chercher le meilleur rendu possible avec les billets et pièces disponibles ;
  • organiser un programme en plusieurs fonctions lisibles.

Toucher-Coule Bateau

Jeu Python de bataille navale sur une grille 10 x 10 avec quatre bateaux, un mode solo, un mode contre une IA et la possibilité de placer soi-meme ses bateaux.

Ce projet m'a permis d'apprendre à :

  • représenter une grille de jeu avec des listes en deux dimensions ;
  • gérer les règles d'un jeu tour par tour ;
  • détecter les tirs rates, les bateaux touches et les bateaux coules ;
  • créer une forme d'IA aleatoire capable de jouer en 1 contre 1 ;
  • vérifier automatiquement les conditions de victoire ou de defaite.

Gestionnaire de Mots de Passe

Programme Python permettant de créer un coffre-fort local protégé par un mot de passe maitre afin d'enregistrer, rechercher, modifier et supprimer des identifiants. Le coffre est chiffre pour éviter de stocker les mots de passe en clair.

Ce projet m'a permis d'apprendre a :

  • comprendre le principe d'un mot de passe maitre ;
  • chiffrer et déchiffrer des données sensibles ;
  • manipuler un fichier JSON pour sauvegarder un coffre local ;
  • génerer des mots de passe forts avec une logique aleatoire sécurisée ;
  • gérer les erreurs comme un mauvais mot de passe ou une recherche inexistante.
